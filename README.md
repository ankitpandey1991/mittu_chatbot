Simple Website Chatbot (HTML, CSS & JavaScript)

A lightweight, beginner-friendly chatbot that can be embedded into any website to interact with visitors, answer common questions, and guide users through your services.

This project is built using pure HTML, CSS, and Vanilla JavaScript — no frameworks or backend required.

Features
 Interactive chat widget UI
 Works fully on the frontend (no server needed)
 Easy to train with custom responses
 Responsive design (mobile friendly)
 Plug-and-play — drop into any website
 Great for FAQs, lead capture, and customer guidance
 Use Cases

This chatbot is perfect for:

Business websites
Portfolio websites
Landing pages
Small e-commerce stores
Customer support FAQ automation

It helps visitors quickly get answers without needing live support.

How the Bot Works

The chatbot uses a simple keyword-matching system.

You can train the bot by editing a JavaScript object that contains:

User questions / keywords
Bot responses

No AI or APIs required — everything runs locally in the browser.
